# Product Brief: FocusBuddy

## Executive Summary

FocusBuddy는 집중력 향상이 필요한 사람들을 위한 macOS 메뉴바 앱입니다. 기존 생산성 앱들이 엄격한 규칙과 죄책감을 통해 동기부여하는 것과 달리, FocusBuddy는 "못해도 괜찮아"라는 따뜻한 철학으로 사용자를 응원합니다.

뽀모도로 등 검증된 집중법을 제공하면서도, 게이미피케이션(칭호 시스템, GitHub 스타일 잔디 캘린더, 95개 이스터에그)을 통해 집중 자체를 즐거운 경험으로 만듭니다. 로그인 없이 완전히 로컬에서 동작하여 개인정보 걱정 없이 사용할 수 있습니다.

---

## Core Vision

### Problem Statement

집중력 저하로 어려움을 겪는 사람들이 많지만, 기존 집중/생산성 앱들은 "엄격한 코치" 방식을 택합니다. 목표 미달성 시 죄책감을 유발하고, 실패를 부정적으로 프레이밍하여 오히려 사용자의 동기를 꺾습니다.

### Problem Impact

- 집중 못하면 죄책감 → 앱 사용 기피 → 더 집중 못함 (악순환)
- 딱딱하고 기능적인 앱 = 지루함 → 사용 중단
- 성취감을 느끼기 어려움 → 지속적인 동기부여 실패

### Why Existing Solutions Fall Short

| 기존 앱 특징 | 문제점 |
|------------|--------|
| 엄격한 타이머 | 실패 시 죄책감 유발 |
| 숫자/통계 중심 | 재미 요소 부재 |
| 소셜/비교 기능 | 다른 사람과 비교하며 자책 |
| 로그인 필수 | 개인정보 노출 우려 |

### Proposed Solution

FocusBuddy는 "응원하는 집중 친구"입니다:
- **메뉴바 상주**: 항상 접근 가능, 방해되지 않음
- **검증된 집중법**: 뽀모도로 등 다양한 집중 기법 제공
- **따뜻한 메시지**: "5분도 대단해!", "다시 와줬네!" 등 긍정 피드백
- **게이미피케이션**: 칭호 수집, 잔디 캘린더, 95개 이스터에그
- **실패도 보상**: 포기해도 이스터에그 획득 가능

### Key Differentiators

1. **철학적 차별화**: "못해도 괜찮아" - 죄책감 없는 집중 도우미
2. **게이미피케이션**: 95개 이스터에그, 칭호 도감, 잔디 캘린더로 모으는 재미
3. **프라이버시 퍼스트**: 로그인 없음, 완전 로컬 데이터, 개인정보 불필요
4. **UX 디테일**: 레벨업 이펙트, 컨페티, 귀여운 캐릭터로 즐거운 경험

## Target Users

### Primary Users

#### 페르소나 1: "집중력 떨어진 민수" (개발자)

**프로필:**
- 28세 스타트업 백엔드 개발자
- 재택근무 3년차, 집에서 집중하기 어려움
- SNS, 유튜브 등 딴짓에 자주 빠짐

**현재 고충:**
- "딱 10분만 유튜브 보려다가 2시간이 지나있음"
- 기존 뽀모도로 앱 써봤지만 타이머 울릴 때 죄책감만 느껴서 삭제
- 집중 못하면 자책 → 더 집중 못하는 악순환

**FocusBuddy와의 만남:**
- 메뉴바에서 살짝 시작, 부담 없음
- 5분만 해도 "대단해!"라고 해주니 기분 좋음
- 칭호 모으는 재미에 "오늘도 한 번 더 해볼까?" 동기부여

**성공 비전:**
- "집중 = 괴로움"에서 "집중 = 게임"으로 인식 변화
- 잔디 캘린더 채워지는 거 보며 뿌듯함

---

#### 페르소나 2: "번아웃 직전 지영" (디자이너/크리에이터)

**프로필:**
- 32세 프리랜서 UI 디자이너
- 마감 스트레스로 번아웃 경험
- 완벽주의 성향, 일 못하면 자책이 심함

**현재 고충:**
- "일해야 하는데 몸이 안 움직여"
- 생산성 앱들의 통계 보면 자괴감 ("이것밖에 못했어?")
- 다른 사람과 비교되는 기능 싫음

**FocusBuddy와의 만남:**
- "못해도 괜찮아" 메시지에 위로받음
- 포기해도 이스터에그 나오니까 "실패"가 없음
- 나만의 속도로, 비교 없이 사용

**성공 비전:**
- 자기 자책 대신 자기 응원으로 바뀜
- 작은 성취도 인정받는 경험

---

### Secondary Users

#### 간접 사용자: 팀원/가족

- 집중하는 시간 알림 → "지금 집중 중이구나" 인지
- 직접 사용하진 않지만 주 사용자의 집중 환경 개선으로 혜택

### User Journey

| 단계 | 민수의 여정 |
|-----|-----------|
| **발견** | 트위터/개발 커뮤니티에서 "죄책감 없는 집중 앱" 입소문 |
| **온보딩** | 다운로드 → 메뉴바에 귀여운 아이콘 등장 → 첫 집중 시작 |
| **첫 경험** | 25분 뽀모도로 완료 → 컨페티 폭발! + 첫 칭호 획득 |
| **Aha! 순간** | 중간에 포기했는데 "괜찮아, 다시 와줘서 고마워" 메시지 → 감동 |
| **일상화** | 매일 잔디 한 칸 채우는 게 루틴, 칭호 도감 100% 도전 |
| **장기 가치** | 집중이 괴로움 → 게임처럼 즐거움으로 인식 변화 |

## MVP Scope

### Core Features (MVP 필수)

**🔧 기본 기능:**
- macOS 메뉴바 상주 앱
- 뽀모도로 타이머 (25분 집중 + 5분 휴식)
- 집중/휴식 상태 화면 알림
- 세션 시작/중지/포기 기능

**🌱 잔디 캘린더:**
- GitHub 스타일 연간 잔디 뷰
- 일별 집중 기록 표시
- 연속 집중일 카운트

**🏅 칭호 시스템:**
- 기본 칭호 10~15개
- 칭호 도감 (획득/미획득 표시)
- 칭호 획득 조건 (세션 완료, 연속일 등)

**💚 긍정 메시지:**
- "5분도 대단해!", "다시 와줬네!" 등
- 포기해도 응원 메시지
- 상황별 따뜻한 피드백

### MVP Success Criteria

**친구들 반응 체크:**
- [ ] "설치 잘 됐어?" → 설치 성공
- [ ] "잔디 채워봤어?" → 핵심 기능 사용
- [ ] "칭호 뭐 얻었어?" → 게이미피케이션 작동

**기술적 성공:**
- [ ] 메뉴바에서 안정적으로 동작
- [ ] 크래시 없음
- [ ] 데이터 로컬 저장 정상
