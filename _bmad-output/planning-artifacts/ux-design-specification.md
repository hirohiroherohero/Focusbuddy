# UX Design Specification - FocusBuddy

**Author:** Hiro
**Date:** 2026-01-07

---

## 1. Design Philosophy

### 핵심 철학: "못해도 괜찮아"

FocusBuddy의 UX는 **응원하는 친구** 컨셉을 중심으로 설계됩니다.

| 기존 집중 앱 | FocusBuddy |
|------------|------------|
| 엄격한 코치 | 따뜻한 친구 |
| 실패 = 죄책감 | 실패 = 이스터에그 기회 |
| 통계로 압박 | 잔디로 뿌듯함 |
| 완벽주의 강요 | 듬성듬성 OK |

### Design Principles

1. **Warmth First** - 모든 인터랙션에 따뜻함을 담는다
2. **No Judgment** - 어떤 행동도 판단하지 않는다
3. **Celebrate Everything** - 작은 성취도 축하한다
4. **Playful Discovery** - 숨겨진 재미를 발견하게 한다
5. **Minimal Friction** - 집중 시작까지 2클릭 이내

## 5. Interaction Patterns

### 5.1 집중 시작 플로우

```
[대기] → 클릭 "집중 시작"
    ↓
[집중 중] → 25분 타이머 시작
    ↓
    ├── 25분 완료 → [완료 축하] → [휴식]
    │
    └── 포기 클릭 → [응원 메시지] → [대기]
```

### 5.2 칭호 획득 플로우

```
[조건 달성]
    ↓
[팝업 애니메이션] - 칭호 아이콘 등장
    ↓
[축하 메시지] - "🏅 '칭호명' 획득!"
    ↓
[확인] → 도감에 추가됨
```

### 5.3 포기 플로우 (핵심 UX)

```
[포기 버튼 클릭]
    ↓
[응원 메시지] - "괜찮아! 시작한 것만으로 대단해 💚"
    ↓
(이스터에그 조건 체크)
    ↓
[이스터에그 발견?]
    ├── Yes → "🥚 '빠른 포기자' 발견!"
    └── No → 대기 상태로
```

## 6. Animation & Motion

### 애니메이션 원칙
- **Duration:** 0.2s ~ 0.4s (빠르고 경쾌하게)
- **Easing:** ease-out (자연스러운 감속)
- **Purpose:** 축하, 피드백, 전환

### 주요 애니메이션

| 이벤트 | 애니메이션 | Duration |
|-------|-----------|----------|
| 타이머 시작 | 버튼 → 타이머 fade transition | 0.3s |
| 세션 완료 | 컨페티 파티클 + 바운스 | 1.0s |
| 칭호 획득 | 아이콘 pop + glow | 0.5s |
| 잔디 채우기 | 색상 fade in | 0.3s |
| 탭 전환 | slide + fade | 0.2s |

### 컨페티 효과 (Phase 2)
- 세션 완료 시 화면 상단에서 컬러풀한 파티클 떨어짐
- 3초간 지속 후 자동 소멸

## 10. Error States & Edge Cases

### 에러 상태

| 상황 | 처리 |
|-----|------|
| 데이터 로드 실패 | "잠깐, 데이터를 불러오는 중..." + 재시도 |
| 저장 실패 | 자동 재시도 3회, 실패 시 알림 |
| 타이머 오류 | 자동 복구, 로그 저장 |

### Edge Cases

| 상황 | 처리 |
|-----|------|
| 자정 넘김 | 현재 세션은 시작일 기준으로 기록 |
| 앱 강제 종료 | 진행 중 세션은 취소 처리 |
| 맥 슬립 | 타이머 일시정지, 깨어나면 재개 확인 |

## 11. Tone of Voice

### 메시지 톤 가이드라인

**DO:**
- 친근한 반말 ("대단해!", "잘했어!")
- 이모지 적극 활용
- 짧고 임팩트 있게
- 무조건 긍정적

**DON'T:**
- 판단하는 표현 ("겨우", "아직", "부족")
- 비교하는 표현 ("다른 사람은...")
- 길고 설교조
- 통계로 압박

### 메시지 예시

| 상황 | ❌ 하지 말 것 | ✅ 이렇게 |
|-----|-------------|----------|
| 5분 포기 | "5분밖에 못했네요" | "5분도 시작이야! 💚" |
| 첫 완료 | "첫 번째 세션 완료" | "우와 첫 집중! 🎉" |
| 재방문 | "돌아오셨군요" | "다시 와줬네! 🙌" |
