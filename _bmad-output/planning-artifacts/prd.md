# Product Requirements Document - FocusBuddy

**Author:** Hiro
**Date:** 2026-01-07
**Version:** 1.0

---

## Executive Summary

FocusBuddy는 집중력 향상이 필요한 사람들을 위한 macOS 메뉴바 앱입니다.

기존 생산성 앱들이 엄격한 규칙과 실패에 대한 죄책감으로 동기부여하는 것과 달리, FocusBuddy는 **"못해도 괜찮아"**라는 따뜻한 철학으로 사용자를 응원합니다. 뽀모도로 타이머와 함께 게이미피케이션(칭호 시스템, GitHub 스타일 잔디 캘린더, 이스터에그)을 통해 집중 자체를 즐거운 경험으로 만듭니다.

로그인 없이 완전히 로컬에서 동작하여 개인정보 걱정 없이 사용할 수 있습니다.

### What Makes This Special

1. **철학적 차별화**: "못해도 괜찮아" - 실패해도 괜찮은 유일한 집중 앱
2. **역설적 보상**: 포기해도 이스터에그 획득 가능 - 실패가 없는 경험
3. **수집의 재미**: 95개 이스터에그, 칭호 도감, 잔디 캘린더
4. **프라이버시 퍼스트**: 로그인 없음, 완전 로컬, 개인정보 불필요

## User Journeys

### Journey 1: 민수 - 유튜브 늪에서 탈출하기

민수는 28세 스타트업 백엔드 개발자다. 재택근무 3년차, 집에서 코딩하다 보면 어느새 유튜브를 보고 있다. "딱 10분만"이 2시간이 되고, 하루가 끝나면 자책한다. 예전에 뽀모도로 앱을 써봤지만, 25분 버티지 못하고 타이머가 울릴 때마다 "또 실패했네" 하는 기분이 들어서 삭제했다.

어느 날, 친구가 "이거 써봐, 좀 다르더라" 하며 FocusBuddy를 보내준다. 메뉴바에 귀여운 아이콘이 생기고, 민수는 별 기대 없이 첫 집중 세션을 시작한다.

25분이 지나고 타이머가 끝났을 때, 화면에 컨페티가 터지면서 "🏅 첫 집중 완료! '신입 집중러' 칭호 획득!" 메시지가 뜬다. "오? 이런 거 있어?" 민수는 피식 웃는다. 다음 날, 또 해본다. 잔디 캘린더에 초록 칸이 하나 채워진다.

일주일 후, 민수는 매일 아침 FocusBuddy를 켜고 시작하는 게 루틴이 됐다. 잔디가 7일 연속 채워졌을 때 "🔥 일주일 연속! '꾸준함의 시작' 칭호 획득!" - 민수는 친구한테 스크린샷을 보낸다. "야 나 칭호 3개 모았어 ㅋㅋ"

---

### Journey 2: 지영 - 자책에서 자기응원으로

지영은 32세 프리랜서 UI 디자이너다. 마감 스트레스로 번아웃을 겪었고, 지금도 "일해야 하는데 몸이 안 움직여"라는 날이 많다. 완벽주의 성향이라 집중 못한 날엔 하루종일 자책한다.

친구가 "지영이 맘에 들 것 같아" 하며 FocusBuddy를 추천한다. 첫 세션, 10분 만에 포기한다. 그런데 화면에 "괜찮아! 10분도 시작한 것만으로 대단해 💚" 메시지가 뜬다. 지영은 순간 멈칫한다.

다음 날, 다시 시도한다. 이번엔 5분 만에 포기. "다시 와줬네! 그것만으로 충분해 🌱". 지영은 조금 웃는다. 세 번째 날, 드디어 25분 완료. "대단해!! 🎉 '포기하지 않는 자' 칭호 획득!"

한 달 후, 지영의 잔디 캘린더엔 듬성듬성 초록 칸이 있다. 완벽하진 않지만, 지영은 그게 오히려 편하다.

---

### Journey 3: 민호 - 포기해도 보상받기

민호는 26세 대학원생이다. 집중력이 너무 짧아서 25분은 커녕 5분도 버티기 힘들다.

FocusBuddy를 설치하고 첫 세션을 시작한다. 3분 만에 "포기" 버튼을 누른다. 그런데... "🥚 '빠른 포기자' 이스터에그 발견! 3분 안에 포기하면 나타나는 숨겨진 보상!"

민호는 어이가 없지만 웃음이 나온다. "뭐야 이게 ㅋㅋㅋㅋ" 다음엔 일부러 2분 만에 포기해본다. "🥚 '더 빠른 포기자' - 와우, 새 기록!"

역설적으로, 민호는 이스터에그가 궁금해서 계속 앱을 킨다. "혹시 1시간 연속하면 뭐가 있을까?" 호기심에 도전해본다. 어느새 25분을 넘긴 민호, "🏅 '진짜 집중러' 칭호 획득!" - 본인도 놀랐다.
