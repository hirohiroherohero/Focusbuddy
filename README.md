## FocusBuddy

macOS 메뉴바에서 동작하는 뽀모도로 기반 집중 타이머 앱입니다.  
기존 생산성 앱처럼 실패를 꾸짖지 않고, **"못해도 괜찮아"**라는 철학으로 사용자를 응원하는 것이 핵심입니다.

-   **플랫폼**: macOS 13.0+ (Ventura 이상), Apple Silicon & Intel 지원
-   **형태**: 메뉴바 상주 앱 (NSStatusItem + NSPopover)
-   **데이터**: 100% 로컬 저장, 네트워크/로그인 없음

### 주요 기능 (MVP)

-   **뽀모도로 타이머**: 25분 집중 / 5분 휴식, 시작·일시정지·포기, 남은 시간 표시
-   **잔디 캘린더**: GitHub 스타일 연간 캘린더, 일별 집중 횟수 및 연속일 카운트
-   **칭호 시스템**: 기본 칭호 10개 내외, 획득 조건 체크 및 도감 UI
-   **긍정 메시지**: 세션 완료/포기/재시작 상황별 응원 메시지
-   **프라이버시**: 로그인/계정 없이 로컬에만 기록

### 개발/실행 방법 (개요)

1. `FocusBuddy/FocusBuddy.xcodeproj`를 Xcode에서 엽니다.
2. 대상(Target)을 `FocusBuddy`로 선택합니다.
3. 실제 사용하려는 Mac을 실행 대상(런 디바이스)으로 선택합니다.
4. Run(⌘+R)으로 빌드 후 실행하면 메뉴바에 아이콘이 표시됩니다.

자세한 요구사항과 기능 범위는 `_bmad-output/planning-artifacts/prd.md` 문서를 참고하세요.
